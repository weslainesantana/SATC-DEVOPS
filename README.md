✅ DevOps

DevOps é uma cultura e conjunto de práticas que integra Desenvolvimento (Dev) e Operações (Ops) para entregar software de forma mais rápida, automatizada e confiável.
Seu foco principal é colaboração, integração contínua (CI/CD), automação e melhoria contínua do processo de entrega.

✅ SRE (Site Reliability Engineering)

SRE é uma abordagem criada pelo Google que aplica princípios de engenharia de software à área de operações, com foco em confiabilidade e estabilidade de sistemas.
Utiliza métricas como SLI, SLO e error budget para equilibrar inovação e disponibilidade.
